# Lab 4 - Esqueleto de Proyecto Node.js/Express

Template o laboratorio académico con la estructura inicial de un proyecto backend utilizando Node.js y Express. Incluye rutas planificadas para la gestión de **productos** y **clientes**, con archivos fuente vacíos listos para implementar.

## Propósito

Este repositorio sirve como punto de partida para:
- Aprender la estructura básica de una aplicación Express.
- Implementar rutas REST para productos y clientes.
- Organizar un proyecto backend en rutas separadas.

## Estructura

```
lab4/
├── routes/
│   ├── productos.js     # Rutas de productos (pendiente de implementar)
│   └── clientes.js      # Rutas de clientes (pendiente de implementar)
├── public/              # Archivos estáticos (futuro)
├── app.js               # Punto de entrada del servidor
└── package.json
```

## Tecnologías

- Node.js
- Express

## Instalación y Ejecución

```bash
npm install
node app.js
# Servidor en http://localhost:3000 (por definir)
```
